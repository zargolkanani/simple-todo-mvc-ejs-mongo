# Simple TODO (MVC + EJS + MongoDB + Caching)

یک پروژه آموزشی ساده برای نمایش ساختار MVC با استفاده از Node.js، Express، EJS، MongoDB Atlas و یک Caching ساده با `node-cache`.

## امکانات
- معماری MVC
- EJS templates + express-ejs-layouts
- اتصال به MongoDB Atlas
- CRUD کامل (ایجاد، خواندن، ویرایش، حذف)
- فیلتر بر اساس وضعیت (pending/done) و جستجو
- Caching ساده با `node-cache` برای بهتر شدن سرعت خواندن
